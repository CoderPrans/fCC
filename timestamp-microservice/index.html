A TimeStamp Microservice

This is a project for freeCodeCamp Certification that requires to create an api endpoind at:

"/api/timestamp/:date_string"

Example Usage:

[project url]/api/timestamp/2015-12-25

[project url]/api/timestamp/1450137600

Example Output:
{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}
